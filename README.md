# Coin Collection Game (Java Swing)

A simple 2D coin‑collection game built with **Java** and **Java Swing**, demonstrating object‑oriented programming, game loop logic, collision detection, and interactive UI rendering.  
This project was developed as part of my learning in Java and game development fundamentals.

---

## 🎮 Game Summary

You control a player character that moves around the screen collecting coins while avoiding enemies.  
The game includes:

- Smooth player movement using keyboard input  
- Randomly spawning coins  
- Automated enemy movement  
- Collision detection (player–coin, player–enemy)  
- Score tracking  
- Game over and restart functionality  
- Continuous game loop using Swing’s `Timer`

## 📁 Project Structure
src/
└── coingame/
├── Main.java        # Launches the game window
├── Panel.java       # Game loop, rendering, input handling
├── Sprite.java      # Base class for all game objects
├── Player.java      # Player movement logic
├── Enemy.java       # Enemy movement logic
├── Coin.java        # Collectible items


---

## ▶️ How to Run

1. Clone or download the repository  
2. Open the project in **Eclipse** or any Java‑compatible IDE  
3. Navigate to:  
   `src/coingame/Main.java`  
4. Run the file as a **Java Application**  

The game window will open automatically.
